1. 현재 페이지 문제점과 개선 방향
문제 1. 기능 키워드가 너무 많다

현재 상품명은 장점이 많다는 인상은 주지만, 첫 1초 안에 무엇을 기억해야 하는지가 분명하지 않습니다. 고객은 긴 키워드 나열보다 대표 효익 한 줄을 먼저 기억합니다.

문제 2. 제품 사용 상황이 선명하지 않다

토너패드이긴 하지만 닦토용인지, 진정팩용인지, 메이크업 전 정돈용인지가 첫 화면에서 명확히 정리되어야 전환력이 높아집니다.

문제 3. 리뷰 강점이 카피로 승격되지 않았다

실사용자 반응에서 보이는 부드러움, 두께감, 밀착력, 쿨링감이 페이지 핵심 카피로 크게 살아나지 않으면 경쟁 제품과 차별이 약해집니다.

개선 방향 요약
이 상품은 “강하게 닦아내는 패드”가 아니라 “예민하고 흔적이 신경 쓰일 때 손이 가는 진정·밀착 PDRN 패드”로 포지셔닝하는 것이 맞습니다.
2. 상품명 개선안
현재 방향

현재 제목은 장점이 많지만, 키워드가 길게 이어져 대표 메시지 집중도가 떨어집니다. 따라서 제품 타입 → 핵심 효익 → 차별 포인트 순으로 재정렬하는 것이 좋습니다.

7. 우선 실행안
1상품명 정리

장점 나열형 제목을 줄이고 跡ケア / 密着 / ひんやり / 敏感肌 중심으로 재구성합니다.

2메인 썸네일 재설계

제품 컷 + 대표 카피 1개만 남기고, 성분/배지/혜택 정보는 서브 썸네일로 이동합니다.

3상세 첫 화면 재작성

성분보다 먼저 누가 왜 쓰는지를 사용 장면 중심으로 설계합니다.

4리뷰 표현 차용

부드러움, 두께감, 늘어남, 쿨링감을 상세카피에 직접 반영합니다.

5톤 조절

치료 이미지보다는 민감 피부를 편하게 정돈하는 데일리 케어 톤으로 안정화합니다.
