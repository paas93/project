가장 큰 문제는 “제품 정체성”이 첫 화면에서 한 번에 읽히지 않는 점입니다.

  • 상품명에 크림 / 인토너 / 미스트가 동시에 들어가 있어 제품 형태가 즉시 이해되지 않음

  • 상세 상단에서 언제, 왜, 어떤 상황에서 쓰는 제품인지 설명이 더 강해야 함

2. 현재 페이지의 강점과 약점

강점

  • 리뷰와 평점이 이미 확보되어 있어 신뢰도 기반은 갖춰져 있음
  • 미스트 입자감, 보습감, 메이크업 위 사용성, 저자극 이미지가 리뷰 장점으로 연결될 가능성이 큼
  • 60ml 휴대성 포인트가 있어 데일리/외출용 소구에 적합함

약점

  • 상품명이 검색 키워드를 많이 넣은 대신, 제품 타입 인지가 약해짐
  • 메인 썸네일이 복잡하여 클릭 전에 핵심 메시지가 묻힘
  • 상세페이지 초반부에서 “토너보다 촉촉하고 크림보다 간편한 미스트”라는 포지셔닝이 선명하지 않음

3. 상품명 개선 방향

정리 원칙: 제품 타입 → 핵심 효익 → 차별 포인트 순으로 재배치

지금처럼 키워드를 많이 넣기보다, 일본 고객이 바로 이해하는 クリームミスト / 保湿ミスト / ミスト化粧水를 앞에 배치하는 것이 좋습니다.

4. 썸네일 개선 방향

핵심 원칙: 예뻐 보이는 것보다, 한눈에 이해되는 것을 우선
